Macam-Macam Perintah di MySQL beserta Fungsinya
Afrizal Meka MSep 15, 2019 at 9:48 pm
share

Kali ini kami MulyanaPedia akan membahas artikel mengenai macam macam atau jenis jenis perintah dasar di MySQL dan Fungsinya, perintah pada MySql sendiri berfungsi untuk menghubungkan suatu program ke Database. Pada dasarnya perintah di dalam MySql sendiri biasanya dikelompokan menjadi tiga kelompok yaitu Data Manipulation Language (DML), Data Definisi Language (DDL), dan Data Control Language (DCL), berikut di bawah ini beberapa contoh perintah dari ketiga jenis perintah di atas.

A. Query Dasar MySql Data Manipulation Language (DML)

1. Insert
Perintah Insert berfungsi untuk memasukan data kedalam sebuah kolom, penulisan query insert adalah sebagai berikut Insert Into Nama_Tambel values (‘data_kolom1′,’data_kolom2′,’….’) keterangan dari perintah disamping : Nama_Tabel adalah nama tabel yang akan kita masukan datanya, lalu data kolom merupakan isi data milik kalian yang ingin di inputkan, Contoh Penggunaan Query Insert Insert Into Mobil values (‘1′,’2010′,’Jaguar’) perintah tersebut ingin menambahkan data kedalam tabel Mobil yang memiliki kolom id, tahun pembuatan mobil, dan nama mobil.

2. Select
Perintah select ini berfungsi untuk menampilkan data dari suatu database, penggunaan query dasar select adalah Select nama_kolom from nama_tabel, misal jika kita ingin menampilkan data dari tabel mobil maka bisa kita tuliskan saja select * from mobil.

3. Update
Perintah update berfungsi memperbarui data yang telah ada, penulisan query dasar mysql update adalah sebagai berikut Update nama_tabel set nama_kolom = data_baru where kondisi, misal kita akan mengubah nama mobil jaguar menjadi lambo, maka bisa kita tuliskan saja Update mobil set nama_mobil = lambo where nama_mobil = jaguar.

4. Delete
Sudah sangat jelas perintah yang satu ini merupakan perintah dasar di mysql yang berfungsi untuk menghapus data di dalam suatu table yang ada. Penulisan perintah delete adalah sebagai berikut Delete Nama_kolom from tabel where kondisi, misal kita ingin menghapus data nama mobil yang memiliki id angka 1 maka bisa kita tuliskan delete nama_mobile from mobil where id=1.

B. Perintah Dasar Mysql Data Definisi Language (DDL)

1. Create
Perintah create ini merupakan suatu perintah dasar di dalam mysql yang memiliki banyak fungsi diantaranya adalah membuat database, membuat view, membuat index, dan juga membuat tabel. Penulisan perintah adalah Create jenis_data_akan_dibuat nama_data, contoh jika kita ingin membuat data base dengan nama db, maka bisa kita tuliskan CREATE DATABASE db.

2. Drop
Sama dengan perintah create, query drop pada mysql ini juga memiliki banyak fungsi, hanya saja jika create untuk membuat maka drop untuk menghapusnya, penulisan perintah drop adalah sebagai berikut Drop jenis_data_didrop nama_data, contoh jika kita ingin menghapus database db bisa saja kita gunakan perintah Drop Database db.

3. Alter
Lalu perintah dasar DDL Data Definisi Language di Mysql selanjutnya adalah Alter, alter ini berfungsi untuk mengubah komponen di dalam suatu tabel, contoh kita bisa menambahkan kolom kedalam suatu data base yang sebelumnya telah di buat jika menggunakan perintah Alter, bisa kita tuliskan ALTER TABLE Nama_Table ADD Kolom_baru type_data, contoh jika kita ingin menambahkan kolom id di table Mobil, bisa kita tuliskan ALTER TABLE MOBIL ADD id varchar(20).

C. Perintah Atau Query MySql Data Control Language (DCL)

1. Grant
Perintah dasar mysql Grant ini berfungsi untuk memberikan suatu hak akses kepada seorang user di dalam data base yang ada, penulisan script grant adalah Grant Hak_akses on database.kolom to ‘nama_user’@’host_name’, contoh kita akan memberikan semua hak akses dan di semua data base yang ada kepada user super yang memiliki host name public (%) maka bisa kita tuliskan Grant ALL on *.* to ‘super’@’%’.

2. Revoke
Jika query dasar mysql Grant memiliki fungsi untuk memberikan suatu hak akses yang ada kepada user di dalam data base, maka Revoke merupakan kebalikan dari grant yang berfungsi untuk mencabut hak akses yang di miliki oleh user di suatu data base, penulisan perintah revoke adalah Revoke Hak_akses on database.kolom from nama_user, contoh jika kita ingin menghapus semua hak akses user super di semua database bisa kita gunakan perintah seperti ini REVOKE all ON *.* from super.
